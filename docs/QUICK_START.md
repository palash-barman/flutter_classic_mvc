# ⚡ QUICK START - 5 Minute Setup

Get the project running in **5 minutes**.

---

## 📋 Prerequisites

```bash
flutter --version         # Ensure 3.10.0+
dart --version           # Ensure 3.0.0+
```

If not installed, download from [flutter.dev](https://flutter.dev)

---

## 🚀 Setup (Copy-Paste)

### **Step 1: Clone**
```bash
git clone <repository-url>
cd flutter_classic_mvc
```

### **Step 2: Install**
```bash
flutter clean
flutter pub get
```

### **Step 3: Run**
```bash
flutter run
```

---

## ✅ Verify Installation

```bash
# Should show 0 issues

# Should pass
flutter test
```
